# App icons

These are placeholder icons generated locally (simple cottagecore terracotta
mark) so the bundle config has real files to point at. Swap them for a
proper design whenever you like — once the Rust toolchain is installed,
regenerate the full set from a single 1024×1024 source image with:

```
npm run tauri icon path/to/your-logo.png
```

That command writes every size this folder currently has, plus `.ico` and
`.icns`, in one step.
